# TO DO LIST

Небольшой pet-проект для портфолио.
Приложение сделано, чтобы показать навыки работы с React и TypeScript на примере todo-менеджера.

## Что реализовано

- Создание и удаление проектов
- Редактирование названия и описания проекта
- Создание, выполнение и удаление задач
- Фильтрация задач (все, завтра, неделя, просроченные)
- Переключение dark/light темы
- Инлайн-редактирование
- Адаптивный интерфейс
- Сохранение данных в `localStorage`

## Технологии

- React
- TypeScript
- Vite
- Zustand
- Ant Design
- React Router

## Запуск локально

```bash
pnpm install
pnpm dev
```

## Сборка

```bash
pnpm build
```

```txt
src/

```

## Заметка

Проект не претендует на production-решение и сделан в демонстрационных целях для портфолио.
